# WIA1003 Computer Systems and Organization (University of Malaya)

This repository contains materials related to the course **WIA1003 Computer Systems and Organization**, offered at the University of Malaya.

This course covers the introduction to computer systems and organization which includes number system, Boolean Algebra, basic logic gates, function simplification, combinational circuit, latches and flip-flop, sequential circuit and addressing mode.
